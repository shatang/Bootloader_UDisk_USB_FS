/********************************************************************************
* 函数名 : IAP_UDisk_USB_FS
* 功  能 : IAP功能，使用U盘Host，采用USB_FS方式进行升级
* 说  明 : 基于Stm32F407
* 入  参 : none
* 返  回 : none
* 设  计 : Shatang                    时  间 : 2020.09.21
* 修  改 : none                       时  间 : none
********************************************************************************/
1. 该IAP升级，通过读取U盘的文件进行APP升级
2. 文件名、类型：Shatang.bin；路径是U盘的家目录，且不支持中文文件名(删减过FatFs库的字库)
3. 该Bootloader进行升级需要重新上电启动
4. 上电2s检测是否插入U盘，2s后检测不到U盘、U盘内对应文件，或升级完毕)进入App程序
5. 该程序运行LED1亮；识别到U盘，且有相关文件LED2亮；擦除芯片FLash完成进行烧录升级文件LED3亮
6. Bootloader程序总大小为26KB
